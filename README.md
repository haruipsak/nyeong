# 고양이 사료 검색 홈페이지
## 제작 배경
- 고양이 사료를 검색할 때 각종 정보가 너무 흩어져있어 선택에 불편함을 느낌. 정보를 한 곳에 모아두는 페이지가 필요하다고 생각하여 개발
